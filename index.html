Smart Parking App *

Javascript, NodeJS, MySQL

Prototype Smart Parking Mobile App using Google Maps API.

Smart Parking aims to reduce congestion caused by circling drivers by helping them find parking quicker. This is a prototpye of a mobile app for a Smart Parking System which uses sensors in parking spaces to detect whether it is occupied or not. The system allows for drivers to search and view available parking spaces. This project was completed for Comp Sci 2206 Software Engineering Workshop II in 2018.

Travelling Thief *

C++

Solution to The Travelling Thief problem.

The Travelling Thief problem is a combination of the Travelling Salesman and Knapsack problem. A thief has to find the optimal route to take and which items to steal along the way - having to deal with the weight along the way. The provided paper was the basis for our solution to the problem. This project was completed for Comp Sci 2203 Problem Solving & Software Development in 2018.
