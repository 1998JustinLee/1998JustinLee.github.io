Smart Parking App

Javascript, NodeJS, MYSQL Github

Prototype Smart Parking Mobile App using Google Maps API.
Smart Parking aims to reduce congestion caused by circling drivers by helping them find parking quicker. This is a prototpye of a mobile app for a Smart Parking System which uses sensors in parking spaces to detect whether it is occupied or not. The system allows for drivers to search and view available parking spaces. This project was completed for Comp Sci 2206 in 2018.
